# Marble Shell Theme (Filled – Blue)

This setup uses the **Marble Shell Theme (Filled – Blue variant)** on **Fedora GNOME**.

---

## Description

**Marble Shell** is a modern GNOME Shell theme available on GNOME Look.  
In this setup, the **Filled – Blue** variant is used.

---

## Installation Steps

1. Download the **Marble Shell Theme** as a ZIP file from  
   [GNOME Look – Marble Shell](https://www.gnome-look.org/p/1977647/)

2. Extract the ZIP archive.

3. Open the extracted folder.

4. Go to the extracted directory and Copy the **blue filled** two folders (`blue light` and `blue dark`) theme folder into the `~/.local/share/themes` directory.

5. Open GNOME Tweaks Go to Appearance Select Marble Shell Filled Blue under Shell
