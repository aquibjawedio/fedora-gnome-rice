# WhiteSur Icon Theme (Alternative Icons)

This setup uses the **WhiteSur Icon Theme** with **alternative (-a) icons** on **Fedora GNOME**.

---

## Description

**WhiteSur Icon Theme** is a macOS Big Surâ€“like icon theme for Linux desktops.  
The **alternative icons (-a)** option installs redesigned macOS default icons.

---

## Installation Steps

1. Download the **WhiteSur Icon Theme** as a ZIP file from [Download](https://github.com/vinceliuice/WhiteSur-icon-theme/archive/refs/heads/master.zip).
2. Extract the ZIP archive.
3. Open a terminal and go to the extracted folder.
4. Run the following command:

```bash
./install.sh -a
```
5. Now go to gnome tweaks and select icons and you are done.
